# Nasdaq-100 Monthly Point-in-Time Momentum Grid Backtest

Run the project to generate the full README tables:

```bash
pip install -r requirements.txt
python run_all.py
```

The generated README will list 3-, 4-, 5-, 6-, and 7-month momentum strategies separately. Each section includes yearly compounded returns, benchmark comparison vs QQQ, summary statistics, and latest monthly selections.
